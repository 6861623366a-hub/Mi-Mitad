# Roadmap Mi Mitad 🚀

## Fase 1: MVP (2-3 semanas)
- [ ] Autenticación con Firebase
- [ ] Generar/unir códigos de pareja
- [ ] Lienzo de dibujo básico
- [ ] Sincronización WebSocket
- [ ] Pantalla de suscripción
- [ ] Sistema de trial de 2 meses

## Fase 2: Monetización (1-2 semanas)
- [ ] Integración Google AdMob
- [ ] Integración Mercado Pago
- [ ] Bloqueo de features para usuarios sin suscripción
- [ ] Dashboard de administrador

## Fase 3: Mejoras (2-3 semanas)
- [ ] Notificaciones push
- [ ] Historial de dibujos
- [ ] Galería compartida
- [ ] Efectos visuales
- [ ] Temas oscuro/claro

## Fase 4: Escalado (Continuous)
- [ ] Optimización de performance
- [ ] Soporte multiidioma
- [ ] Análisis y métricas
- [ ] Marketing y crecimiento
- [ ] Nuevas features según feedback

## Entregables

### Corto plazo (1 mes)
- ✅ Repo estructurado
- ⏳ App funcional en desarrollo
- ⏳ Beta testers

### Mediano plazo (3 meses)
- Lanzamiento en Play Store
- Lanzamiento en App Store
- 1000+ descargas

### Largo plazo (6+ meses)
- Monetización activa
- Comunidad de usuarios
- Features avanzadas
